# Assets Folder

This folder contains visual assets for the System Design Zero to Hero repository.

## Contents

- **logo.svg** - The official logo for the project (SVG format for scalability)
- **logo.png** - Alternative PNG version of the logo (if needed)
- **banner.png** - Repository banner image (if created)
- **icons/** - Custom icons for documentation (if needed)
- **screenshots/** - Screenshots of diagrams or examples (if needed)

## Usage

Assets in this folder are referenced in the main README.md and other documentation files.

### Logo Usage

The logo can be embedded in markdown files using:

```markdown
<img src="assets/logo.svg" alt="System Design Zero to Hero Logo" width="200"/>
```

## Guidelines

When adding new assets:

1. Use descriptive filenames
2. Optimize images for web (compress when possible)
3. Use SVG format for logos and icons when possible
4. Use PNG for screenshots and photos
5. Keep file sizes reasonable
6. Update this README when adding new asset types

## Design Specifications

### Logo
- Format: SVG (scalable vector graphics)
- Dimensions: 200x200px
- Colors: Blue theme (#2563eb primary, #60a5fa accent)
- Background: Rounded corners (20px radius)
