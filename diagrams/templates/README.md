# Diagram Templates

This folder contains reusable templates for creating consistent system design diagrams.

## Available Templates

Templates help maintain visual consistency across all diagrams.

### Basic Components

Common system design components that can be reused:
- Server icons
- Database icons
- Cache icons
- Load balancer icons
- Client device icons
- Network/cloud icons

### Template Types

1. **Basic Architecture Template**
   - Single server + database setup
   - Good starting point for simple designs

2. **Microservices Template**
   - Multiple services with API gateway
   - Message queue integration
   - Service discovery

3. **Scaled Architecture Template**
   - Load balancer
   - Multiple servers
   - Database replication
   - Caching layer

4. **Distributed System Template**
   - Multiple data centers
   - CDN integration
   - Global load balancing

## How to Use Templates

1. Open template file in your diagram tool
2. Save a copy with a new name
3. Modify for your specific use case
4. Maintain the color scheme and style
5. Export to appropriate phase folder

## Creating New Templates

When creating templates:
- Keep them generic and reusable
- Use the standard color scheme
- Include a legend
- Document the template in this README
- Save in an editable format (e.g., .drawio)
